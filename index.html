Why Kartograph?

Kartograph is a new framework for building interactive map applications without Google Maps or any other mapping service. It was created with the needs of designers and data journalists in mind.

Please check out the showcase section to see what Kartograph can do for you or dive into the details on how it works.

Current Progress

At the moment Kartograph is still in an early pre-release state, which means that it basically works and is already used a lot by its author but there are some more things left to do to make it as easy-to-use for everybody else. This includes tutorials and step-by-step examples and API documentation.

If you like Kartograph and can’t wait to support its development, please find the donation buttons on the left. Any contribution is much appreciated. Thanks!

Features

  • Supporting beautiful maps™
  • Data vis friendly
  • Tons of map projections, some even in 3D.
  • Runs 100% stand-alone
  • Gracefully degrades to IE7
